Overview

Bookmarks Shortcut ("the Extension") respects your privacy. This privacy policy explains what data we collect, how we use it, and how we protect it.

Data Collection

The Extension collects and stores the following data locally on your device:

  • Shortcut key configurations (keys, URLs, titles, tags, aliases)
  • Action settings (how shortcuts behave)
  • Custom JavaScript scripts (if you choose to create them)
  • Extension preferences (column count, filter position, etc.)

Data Storage

All data is stored in one of two locations:

  1. Local Storage: Data stored only on your current device
  2. Chrome Sync Storage: Data synchronized across your Chrome browsers (only if Chrome sync is enabled)

Important: All data remains within your Chrome browser ecosystem. We do not operate any external servers or databases.

Data Usage

Your data is used exclusively for:

  • Executing shortcut key actions
  • Displaying your bookmarks and shortcuts in the popup
  • Providing search functionality
  • Synchronizing settings across your devices (if Chrome sync is enabled)

Data Sharing

We do not share, sell, or transmit your data to any third parties.

  • No data is sent to external servers
  • No analytics or tracking services are used
  • No personal information is collected beyond what you explicitly configure

Permissions Explanation

The Extension requires the following permissions:

  • tabs: To manage and switch between browser tabs
  • storage: To save your shortcut configurations locally and in Chrome sync
  • contextMenus: To add "Add to Bookmarks Shortcut" to right-click menu
  • userScripts: To execute custom JavaScript scripts you create
  • host_permissions: <all_urls>: To access URLs you specify and execute scripts on those pages

These permissions are used only for the functionality described above and never for data collection or tracking.

Open Source

The Extension is open source. You can review the complete source code at:
https://github.com/itohenthunagi/bookmark-shortcutkey

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated "Last updated" date.
